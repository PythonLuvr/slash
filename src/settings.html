/settings

Search

Search engines
Every available engine. Set your default (used in the address bar) and star the ones that appear as quick picks on the start page.

Default browser

Make Slash your default browser
Open web links in Slash.

Move your stuff to Slash

Import from another browser
Bring your bookmarks, history, and signed-in sessions over from a browser on this computer. Everything is read locally and stays on your machine. Cookies from very recent Chrome may be protected by app-bound encryption and can't be moved.

Passwords

Saved logins
Stored encrypted on this device and filled only on the site they belong to.

Privacy & security

Block ads & trackers
EasyList + EasyPrivacy filter lists, applied in the network layer.
HTTPS-only mode
Upgrade sites to HTTPS and warn before loading over plain HTTP.
DNS over HTTPS
Encrypt DNS lookups so your network and ISP can't read them. Turn off if a network blocks it.
Clear browsing data
Remove what Slash has stored on this device. Clearing cookies signs you out of sites.

Extensions

Chrome extensions
Install extensions straight from the Chrome Web Store, open it and click "Add to Chrome". You can also load an unpacked extension folder for development. Extensions reload each launch.

Updates

Check for updates automatically
Offer optional updates when a new version ships. You always choose whether to install; updating restarts Slash in place.

Appearance

Accent color
One restrained accent, used across the UI.

AI

AI providers & keys
Choose Claude, Gemini, or ChatGPT, switch between CLI and API, and add your keys.
